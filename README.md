# Hospital Management System

Hospital Management System is a web-based application designed to manage hospital operations through role-based access. 
The system supports three main user roles: **Administrator**, **Doctor**, and **Patient**, each with specific functionalities.

The application allows users to log in and interact with the system according to their role.

---

## Features

### Authentication
- User login system
- Role-based access control

---

### Administrator
- Manage and view scheduled appointments
- View and manage administrator profile information

---

### Doctor
- View assigned appointments
- View personal profile information
- Access patients' medical records

---

### Patient
- Request appointments
- View personal appointments
- View billing information
- Access personal medical records

---

## Security notes

This project is a learning/demo application.

Password hashing was intentionally omitted to keep the focus on
application structure and role-based access control.
In a real-world deployment, passwords should be securely hashed
